# Html5maker 

Html5maker is a simple interface around xml.etree python module.
This is a html and svg generator. it depends on Markdown (library) to parse markdown and xml.etree

## introduction

I know, there is many web app on python (django, flask) and html generator on github.

The main idea of this repo is to use xml.etree in an easier way and to be pedagogic.
it needs work to be achieved.
